# TRADIOS Phase 2 — Daily Operating System for Tradies

## Goal
Transform TRADIOS from a feature-rich dashboard into a daily operating system that feels effortless, intelligent, and calm. Prioritize workflow quality, speed, and premium UX over new features.

## Features to Build (in order)

### 1. Revenue & Pipeline Analytics
**Location:** Dashboard (replaces static metrics)
- **Revenue chart** — Simple Canvas bar chart showing weekly revenue for last 6 weeks
- **Pipeline value** — Sum of all jobs in "Booked" + "In Progress" columns
- **Quote win rate** — % of quotes that converted to booked jobs
- **Average job value** — Average $ amount across all jobs
- **Monthly target** — Progress bar toward monthly goal
- **Lead velocity** — Leads/week trend

### 2. Workspace Switcher
**Location:** Sidebar top
- Dropdown/select to switch between workspaces
- Each workspace has its own jobs, leads, quotes, photos
- Context-aware data filtering
- Visual indicator showing active workspace
- Different accent colors per workspace (Antonio=red, Prime=blue)
- **Workspaces:** Antonio Paving, Prime Property, All

### 3. Client Timeline
**Location:** Job card click → timeline modal
- Visual timeline showing: Lead Created → Quote Sent → Booked → Site Visit → Completed
- Date stamps for each milestone
- Auto-calculated duration between stages
- Key milestones highlighted with icons
- "Add milestone" button for custom events

### 4. Smart AI Actions (Embedded)
**Location:** Contextual buttons throughout the app
- **On Leads:** "✉️ Draft email with AI" — opens AI panel pre-filled
- **On Jobs:** "🤖 AI quote for this job" — navigates to quotes with pre-filled data
- **On Dashboard:** "📊 Generate weekly report" — AI summarizes all activity
- **On Photos:** "🏷️ Describe this photo" — AI generates description
- All use existing AI_KEY + OpenRouter

### 5. Calendar Integration
**Location:** New "Calendar" section in Operations
- Job calendar showing jobs by date
- Today's jobs highlighted
- Weekly/monthly view toggle
- Quick-add job from calendar
- Click job → see details

### 6. Premium Polish
- Micro-animations on interactions
- Better empty states
- Skeleton loading states
- Haptic-ready button feedback
- Keyboard shortcuts (1-9 for sidebar nav)

## Files Affected
- `index.html` (single file SPA)

## Architecture
- All new features in the same single HTML file
- No external dependencies
- Canvas-based charts (no charting library)
- localStorage for all data persistence
- OpenRouter for AI features

## Size Estimate
- Current: ~107KB
- Target: ~120KB
